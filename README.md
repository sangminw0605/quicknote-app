# QuickNote App

A simple note-taking application that uses browser local storage for persistence.

## Run Locally

Clone the project

```bash
  git clone https://github.com/sangminw0605/quicknote-app.git
```

Go to the project directory

```bash
  cd quicknote-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
